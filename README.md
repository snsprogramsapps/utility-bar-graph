# Project Bar Graph

Mobile-friendly utility bill graph tool for field consultants.

## GitHub Pages Setup

1. Create a new GitHub repository.
2. Upload these files to the repository root:
   - `index.html`
   - `manifest.json`
   - `service-worker.js`
   - `README.md`
3. In GitHub, open **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. Open the GitHub Pages URL on iPhone Safari.
7. Tap **Share** > **Add to Home Screen**.

## Notes

- Camera upload works best from the hosted HTTPS GitHub Pages URL.
- The app caches its main files after the first successful load.
- Export PNG/PDF buttons work in Safari, but iOS may ask where to save or share the file.
